Sprint 1

1. Wat is het nut van een breakdown schets?
Je schets voor jezelf een beter beeld over hoe je het wilt hebben, daarnaast geef je ook aan welke html en css elementen je daarbij wilt gebruiken. Dit helpt je met het coderen.

2. Waar moet je op letten als je een breakdown schets maakt?
Dat je goede html en css elementen op de goede plekken zet en zorg ervoor dat het duidelijk blijft.

3. Wat heb ik vandaag geleerd?
Breakdown schetsen is een belangrijke stap in het proces, je schets je idee schrijft erbij. Daarnaast dat er heel veel elementen zijn die je kan toepassen tijden het coderen. Als je een ! typt en enter dat er een basis code komt. Daarnaast ook de verschillende benamingen, bijvoorbeeld dat alles body heeft en je daarbinnen verschillende sections hebt. Ook dat je moet opletten dat je niet informatie wat in css geschreven moet worden in html.
